> 我打算开发一款web网页游戏合集，包含俄罗斯方块，贪吃蛇，坦克大战。在开始之前我应该提前安装什么skills?

1. develop-web-game
   适合你这个场景，应该是最核心的。你要做俄罗斯方块、贪吃蛇、坦克大战，这类项目通常会涉及 Canvas、键盘输入、游戏循环、
   碰撞检测、资源组织、关卡/状态管理，这个 skill 最贴近需求。
2. playwright
   建议安装。后面你做完页面和游戏逻辑后，需要自动化验证：

- 页面能否正常加载
- 键盘操作是否生效
- 游戏开始/暂停/结束流程是否正常
- 不同分辨率下 UI 是否错位

3. screenshot
   建议安装。它适合做界面验收，尤其是这种带菜单页、游戏卡片、排行榜、暂停弹窗的网页项目，方便快速检查视觉结果。
4. 部署类二选一
   如果你准备上线，建议至少装一个：

- vercel-deploy
- netlify-deploy
  如果你还没确定平台，先不用装两个，后面选一个就行。
  可选的补充项：
- figma
  如果你会先做原型或 UI 设计稿，再开发，值得装。
- security-best-practices
  如果后面会加登录、排行榜、存档同步、用户数据，就可以装；纯前端单机小游戏阶段不是刚需。
- openai-docs
  只有当你准备接入 OpenAI 能力时再装，比如：
  - AI 对手
  - AI 关卡生成
  - AI 提示/陪玩
  - 玩家对话 NPC

  我的建议是你现在先装这一组最实用：

- develop-web-game
- playwright
- screenshot
- vercel-deploy 或 netlify-deploy
